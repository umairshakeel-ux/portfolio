From Clutter to Clarity: A Clean, Modern UX for On-Demand Cleaning Services

  • SaviourVPN is a mobile VPN application designed to help users enjoy safe, private, and unrestricted internet access. The goal was to design a clean, trustworthy, and easy-to-use interface that balances technical performance with a friendly user experience.

Project Context & Problem

Many home-service platforms exist, but users often face friction when booking cleaning services: unclear pricing, confusing scheduling interfaces, and lack of trust or transparency. My client wanted to build a mobile app that:

  • Makes booking a cleaning service as simple as ordering a ride
  • Provides clear pricing, service details and trusted professionals
  • Offers a dashboard for users to manage past bookings, upcoming tasks, and support

Key challenges identified:

  • Simplifying multiple service options (deep cleaning, regular cleaning, add-ons) into an intuitive flow
  • Helping users feel confident about the cleaner’s profile, rating, and reliability
  • Ensuring the mobile interface is fast, clean, and mobile-first (iOS & Android)

My Role & Process

I led the UI/UX design from concept through high-fidelity screens and handoff. My process included:

  • User Research: Defined primary personas (e.g., busy professionals, families with children) and mapped their booking pain points.
  • User Flow Mapping: Mapped out key flows service selection → scheduling → payment → confirmation → dashboard.
  • Wireframes: Created low-fidelity sketches to test layout and flow efficiency.Design System Setup: Developed a scalable Figma component library (buttons, cards, modals, icons) with variants for states (idle/hover/active).

  • High-Fidelity Screens: Designed final mobile UI in Figma with clean spacing, modern typography, and brand-aligned colors.
  • Wireframes: Created low-fidelity sketches to test layout and flow efficiency.
  • Prototyping & Handoff: Delivered interactive Figma prototype for user testing and handed off assets/specs for development.

Service Booking Flow:

  • Clean hero screen with bold CTA: “Book Cleaning Now”
  • Service categories displayed as cards with icons and quick descriptions
  • Transparent pricing shown upfront, plus optional add-ons with toggles
  • Calendar view for scheduling, with availability and filters (time of day, cleaner gender preference).

Cleaner Profiles & Trust:

To create a minimal yet powerful VPN app that allows users to:

  • Each cleaner has a card with photo, rating, number of jobs done, and specialties
  • Option to favorite a cleaner, repeat past bookings or view history
  • Trust badges (background verified, multiple reviews) to boost confidence

Design System & Scalability:

  • Components built with Auto Layout so new service cards or add-ons can be added quickly
  • Light and Dark mode ready
  • Icons and asset library kept modular for future expansion (e.g., cleaning supplies, pet-friendly services)

Key Screens

  • The streamlined booking flow reduced the average number of screens from 6 → 4, making the experience faster and more engaging.
  • Early prototype testing with 5 users from the target demographic showed a task completion rate of 92%, up from 75% in the previous version.
  • Cleaner profile trust elements resulted in a 20% increase in first-time bookings during pilot phase.
  • The design system reduced future screen build time by an estimated 40%, lowering development costs and speeding up rollout of new features.
